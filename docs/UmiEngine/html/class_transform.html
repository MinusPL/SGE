◆ ApplyTransform()

glm::mat4 Transform::ApplyTransform ( )

Returns transformation matrix for object.

This method returns transformation matrix for object after applying position, scale and rotation to the identity matrix. If transform has parrent it is then multiplied by parrent's transformation matrix.

Returns position in world space.
